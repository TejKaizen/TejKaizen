### 📂 Projects I'm proud of

#### 🔐 Subscription Management System
A full backend system for managing user plans, billing cycles, and renewals.

Built with **Flask + MySQL + Python**. Includes RBAC authentication, automated expiry alerts, plan upgrade/downgrade logic, and optimised queries to keep response times low.  
The part I enjoyed most: designing the role-based access system cleanly from scratch.

---

#### 📍 Mechanic Near You
A location-aware app that connects users to nearby mechanics in real time.

Built the entire search and filter system, integrated **Google Maps API** for proximity matching, added a multi-channel notification layer (email/SMS/in-app), and made the UI fully responsive for mobile.  
What I learned: working with live APIs and handling async states gracefully.
